## Goal
Implement `--begin` and `--end` date filtering for register/balance commands in both Python and Rust.

## Acceptance Criteria
- [ ] Python: `muonledger register --begin 2024-03-01` filters to transactions on/after that date
- [ ] Python: `muonledger register --end 2024-03-01` filters to transactions before that date
- [ ] Rust: Same `--begin`/`--end` flags work in CLI
- [ ] Parity tests `date_filter_begin` and `date_filter_end` pass for both

## Notes
Python: --begin/--end flags exist in CLI but produce empty output (filtering not applied). Rust: --begin/--end flags not recognized by clap CLI at all (exit code 2).
